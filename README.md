# 무영생존기 세계관 소개 사이트 v3

## 사용법
1. 이 폴더의 파일 전체를 GitHub 저장소에 올립니다.
2. GitHub 저장소의 Settings → Pages에서 배포 브랜치를 선택합니다.
3. `index.html`이 루트에 있도록 유지합니다.

## 반영 사항
- 붓글씨 스타일 메인 타이틀
- 세계관 / 조직 / 등장인물 / 기본상식 바로가기 버튼
- F1023 무영시 지도와 5개 지구 선택 버튼
- 기타 장소 소개 삭제
- 인물 외 가로 이미지 비크롭 처리
- 조직 순서에 따른 인물 배치 및 백진영 F73 무표정 적용
- 요정가루 농도별 설명 삭제, 용도·합법성·주의사항 추가
- 마지막에 이야기 시작 문구 배치
